== How the ZT grader worked

* *Module 1*: Runner used `kubernetes.core` with a cluster-admin SA token directly — no SSH needed
* *Module 2*: Runner SSHed to bastion and used `ansible.builtin.script` to copy and run a bash script
* *Module 3*: Runner SSHed to bastion and ran `oc --as developer` — the bastion's system:admin can impersonate any user
