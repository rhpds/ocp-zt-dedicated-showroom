= Module 3: As Developer User

Create a Secret in `demo-project` as the `developer` user. The grader uses `oc --as developer` impersonation on the bastion.

== Credentials

[cols="1,2"]
|===
| OCP API URL | `{openshift_api_server_url}`
| Username    | `developer`
| Password    | `{openshift_cluster_admin_password}`
|===

== Your Task

Use the *Terminal (Node)* tab, log in as developer and create a Secret:

[source,sh,role="send-to-wetty",subs="attributes+"]
----
oc login {openshift_api_server_url} \
  -u developer \
  -p {openshift_cluster_admin_password} \
  --insecure-skip-tls-verify

oc create secret generic dev-secret \
  --from-literal=api-key=my-app-key \
  -n demo-project
----

== Verify

Click *Validate* to check that `dev-secret` exists in `demo-project` and is accessible by `developer`.

Click *Solve* to have the grader create it using `oc --as developer`.
