= Module 2: Bash Script on Bastion

Create a ConfigMap in `demo-project` from the bastion terminal. The grader runs a bash script on the bastion as `lab-user` (system:admin).

== Your Task

Use the *Terminal* tab to connect to the bastion, then create the ConfigMap:

[source,sh,role=execute]
----
oc create configmap app-config \
  --from-literal=environment=production \
  --from-literal=app=my-app \
  -n demo-project
----

== Verify

Click *Validate* to check that `app-config` ConfigMap exists in `demo-project`.

Click *Solve* to have the grader run the bash script on the bastion for you.
