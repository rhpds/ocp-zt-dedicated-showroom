= Module 2: Bash Script on Bastion (Testing send-to-wetty)

This module also uses *send-to-wetty*. Confirms the wetty terminal works for multi-command workflows.

Create a ConfigMap in `demo-project` from the bastion terminal.

== Your Task

Use the *Terminal (wetty)* tab — click ▶:

[source,sh,role="send-to-wetty",subs="attributes+"]
----
oc create configmap app-config \
  --from-literal=environment=production \
  --from-literal=app=my-app \
  -n demo-project
----

== Verify

Click *Validate* to check that `app-config` ConfigMap exists in `demo-project`.

Click *Solve* to have the grader run the script on the bastion for you.
